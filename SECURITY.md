# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.4.x   | Yes       |
| < 0.4   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, email: security@revy.fun

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and provide a detailed response within 7 days.

## Smart Contract Security

The RevyRouter contract on Arbitrum has been internally reviewed. If you find a vulnerability in the contract:
- Report via the email above
- Do not exploit the vulnerability
- Contract address: `0x75da9759d9e0a22d9b8a77ec1ec57f99e6759255`

## Scope

- Routing engine (`src/`)
- Smart contracts (`contracts/`)
- API endpoints
- Dependencies with known CVEs
